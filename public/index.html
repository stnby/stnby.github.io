kChat — A functional chat-room server that you can connect using netcat or any TCP client.

A chat-room server that we, along with my colleague have written in C. Why? Because IRC was not quite as good. Drop us a message: nc chat.kernal.eu 1337 or telnet chat.kernal.eu 1337.

SocketBroker — A tool written in C that lets you broker your TCP sockets.

A single-threaded tool written in C that utilizes multiplexing to broker another TCP connection. Check out my NMEA message stream powered by this tool on ais.kernal.eu:5060.

GoMarkovAPI — A simple API that generates double ended Markov chains.

A simple API written in Golang and powered by Redis DB and also Docker containers that generates double ended Markov chains from a given input. (Used as a funny chat bot in my Telegram group)

kCraft — A „Minecraft“ launcher written purely in POSIX shell.

A fully fledged „Minecraft“ client cramped in a single shell script. Handles asset, version downloads and library extraction.

kernal.eu — A website and a theme for Kernal Community powered by Hugo static site generator.

A blog of Kernal Community.

ffsteg — A stenography algorithm based around Farbfeld image format without any dependencies.

University assignment.

AlpineDroid — Alpine Linux chroot environment for your Android smart device.

An easy and quick script to create Alpine Linux chroot environment on your rooted Android device.

Tor Balanced — Few simple scripts to create a multi Tor instance balanced SOCKS5 proxy.

A way to download large files through Tor quickly.

kTrack (WIP) — A system to collect and visualize all sorts of access point information and location. Such as WiFi, BT and LTE towers.
